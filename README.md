# ustwoapp.github.io

Public site for the **UsTwo** iOS app — landing page + legal docs, served via GitHub Pages.

- `/` — landing
- `/privacy/` — Privacy Policy (linked from the app + App Store Connect)
- `/terms/` — Terms of Service

## Publish
1. Create a repo named **`ustwoapp.github.io`** under the `ustwoapp` GitHub account
   (a user/organization Pages repo serves at the root domain).
2. Push this folder to its `main` branch.
3. Repo → Settings → Pages → Source: **Deploy from a branch** → `main` / `/ (root)`.
4. Live at **https://ustwoapp.github.io** within ~1 minute.

The app's Paywall links and App Store Connect "Privacy Policy URL" point to
`https://ustwoapp.github.io/privacy`.
